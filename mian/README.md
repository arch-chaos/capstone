# ExploreWorld Travel Agency Website

## Project Overview
This is a responsive travel agency website built using HTML5 and CSS3, showcasing various tour packages, customer reviews, and an inquiry form. The website demonstrates modern web design principles with semantic HTML structure, CSS Flexbox and Grid layouts, and responsive design techniques.

## Theme
**Travel Agency Website** - A platform for travelers to explore and book curated tour packages to popular destinations around the world.

## Features Implemented

### Layout & Structure
- **Semantic HTML5**: Used proper semantic elements like `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, and `<footer>` for better accessibility and SEO.
- **Responsive Design**: Implemented using CSS Flexbox and Grid with media queries for desktop (≥1024px), tablet (~768px), and mobile (≤480px) breakpoints.
- **Navigation**: Fixed header with logo, navigation links, and mobile-friendly hamburger menu.

### Content Sections
1. **Hero Section**: Large banner with call-to-action button and keyframe animation.
2. **Tour Packages**: Grid layout displaying featured packages with images, descriptions, and pricing.
3. **Customer Reviews**: Testimonials section with review cards.
4. **Inquiry Form**: Contact form with validation for planning trips.

### Visual Design
- **Color Palette**: Consistent use of primary (#007bff), secondary (#28a745), and accent (#ffc107) colors.
- **Typography**: Roboto font family from Google Fonts with proper hierarchy.
- **Hover Effects**: Smooth transitions and hover animations on interactive elements.
- **CSS Variables**: Used for consistent spacing, colors, and sizing throughout the site.

### Accessibility & Performance
- **Alt Text**: All images include meaningful alt attributes.
- **Form Labels**: Properly associated labels for all form inputs.
- **ARIA Labels**: Added where appropriate for screen readers.
- **Skip Link**: "Skip to main content" link for keyboard navigation.

## Technical Details

### CSS Techniques Used
- **Flexbox**: For header layout, footer sections, and form alignment.
- **CSS Grid**: For package cards and review sections.
- **Media Queries**: Responsive breakpoints at 768px and 480px.
- **CSS Variables**: For maintainable and consistent styling.
- **Animations**: Keyframe animation for hero text fade-in effect.

## How to View the Website

### Local Viewing
1. Clone or download the project files.
2. Open `index.html` in a modern web browser.
3. The website is fully responsive and can be viewed on desktop, tablet, and mobile devices.

### Browser Compatibility
- Chrome (recommended)
- Firefox
- Safari
- Edge

## Responsive Breakpoints
- **Desktop**: ≥1024px - Full grid layouts and horizontal navigation
- **Tablet**: ~768px - Single-column grids and stacked header elements
- **Mobile**: ≤480px - Optimized spacing and touch-friendly elements
